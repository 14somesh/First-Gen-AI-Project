## Phase 1 – Data Ingestion

This folder is reserved for assets and documentation specific to **Phase 1: Data Ingestion** of the AI Restaurant Recommendation Service.

Current implementation details:
- Core ingestion code lives in the `ingestion` package:
  - `ingestion/phase1_ingestion.py` implements the end-to-end ingestion flow.
  - `ingestion/models.py` defines the `RestaurantRecord` model.
- Tests reside under `tests/test_phase1_ingestion.py`.

Use this folder for any Phase 1–specific notes, sample outputs, or auxiliary scripts you may want to add later.
